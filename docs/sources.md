# 内容与素材来源记录

本文件记录网站公开内容的来源、制作方式和许可状态，便于后续核验、署名、替换或删除。没有找到可靠记录的项目会明确标为“待补录”，不会据此推定素材可以自由使用。

## 1. 使用原则

- 记录正文、文化拓展、翻译、音频、图片和其他外部资料。
- 至少保留：页面、素材类型、作者或提供者、原始网址或书目信息、许可、获取或生成日期。
- “网上公开可见”不等于“可以任意复制”；“非商业个人学习”说明也不自动替代授权。
- 无法确认来源或许可的素材，应补证、替换或移除。
- 权利人提出异议时，及时定位对应页面和文件，保留处理记录。
